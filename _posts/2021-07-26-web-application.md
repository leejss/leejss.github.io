## Web application(웹 애플리케이션)이란?

### Application Program

사용자 또는 다른 애플리케이션을 위해 특정 기능을 수행하는 프로그램.

### Web Application

웹 브라우저 인터페이스를 가지고 있으며 인터넷을 통하여 제공되는 애플리케이션. 웹사이트, 웹 서비스 등이 이에 속한다.

사용자는 웹 브라우저를 통하여 웹 앱에 접근할 수 있다.

주로 HTML, Javascript, CSS로 만들어진다.
